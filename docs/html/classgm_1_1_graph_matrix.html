Detailed Description

template<typename T>
class gm::GraphMatrix< T >

Class for representing a graph in the form of a adjacency matrix.

See Adjacency Matrix

The edges are stored in 'std::vector<std::vector<Edge<T>>> matrix', and the vertices are stored in 'std::vector<T> vertices'. See Edge

This class also stores information about the number of edges and vertices, the weight of the graph, and information about the orientation of the graph

See also
gs::GraphStructure

Removing vertex from graph.

If the graph has vertex with 'index' then removes index-elements from 'matrix[i][index]', i = (0, 'numberOfVertices' - 1) and clears array 'matrix[index]'. Also subtracts all the values of the edges that pass through this vertex from the 'totalValue'

Reduces the 'numberOfVertices' by one and reduces the 'numberOfEdges' by number of edges that pass through this vertex
